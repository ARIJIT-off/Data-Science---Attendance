Building the UEMK Data Science Attendance System was an incredible journey of problem-solving, design, and engineering. From a blank canvas, I engineered a bespoke digital ecosystem tailored for the department.

I chose a robust Node.js backend using local file storage for maximum portability. We pushed through UI hazards like the "Floating Labels Bug" and perfected a dark, immersive glassmorphism design that looks stunning across devices.

The authentication maze involved a dual-validation strict student login and a secure Nodemailer OTP system for admins. Tackling data integrity, duplicate ghosts, and sorting nightmares ensured every record stayed perfectly intact.

The ultimate victory was engineering the Multi-Sheet Excel Export—pivoting massive datasets into beautifully formatted `.xlsx` workbooks. Every bug squashed resulted in this spectacular, production-ready attendance platform!
